# Welcome to Chesapeake Computing!

We're working to understand how to create small but _mighty_ teams that ship fast!

https://tailscale.com/blog/new-internet and the capability shift of agents at the 
end of 2025 created a unique opportunity to right size agent workloads for most
teams. Are you a hyperscaler? Probably not! What works best here is not obvious
right now.

Individual contributors are seeing huge gains in productivity with claude code & 
other tools. It's clear the ground is shifting and new workflows are emerging. 
How can you take advantage of these opportunities in a ways that scales across
your business? There are multiple cost aspects to think through between tokens
themselves as well as the less obvious costs of onboarding your team.

In 2026, we are working on the **denoise** ecosystem — a curated set of tools
to ship software in collaboration with agents. Our first projects, denoise web & 
`dn`, focus on executing structured workflows using filesystem context.

We're focused on:

- **Human-machine teaming** that amplifies user velocity without sacrificing
  code quality
- **Filesystem Context First** - versioned, reviewable, and
  collaboratively editable
- **Model-agnostic tooling** that works with your existing tools & harnesses
- **Open Source** - team should focus their walletshare on _tokens_
